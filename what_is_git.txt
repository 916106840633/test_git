what is git?

在文件夹内启动命令行并且使用命令git init
那么git就初始化这个文件夹为仓库，就可以开始管理这个文件夹内的所有文本文件

git可以看成给这个文件夹独立的增加一个管理程序
git不是一个软件
它依附于一个文件夹形成相应的管理程序
它是和这个文件夹绑定的，而不是一个系统软件管所有文件夹

一般流程
文件有改动（修改 或者 新增） //如果是删除需要使用命令 git remove
git add * 加入到暂存区
然后使用
git commit -m "" 统一加入仓库

有关远程仓库的使用
首先是要执行一次远程仓库的关联
git remote add .......
然后可以使用
git pull ... 从仓库拉去内容
git push ... 给仓库推送文件

中间可能存在冲突问题，具体参考aboutConflict.txt
